# Normalizer
OrderNormalizer is an intelligent data-processing engine designed to transform unstructured purchase requests into standardized entries mapped directly to the KTU Directory (Catalog of Goods, Works, and Services).

OrderNormalizer (Russia KTU edition)
🚀 Overview
The tool eliminates "data noise" by converting free-form text into structured, compliant data. It acts as a bridge between informal human requests and rigid database requirements.
🛠 Core Logic
Ingestion: Receives non-formalized requests (e.g., "Need 10 boxes of A4 paper").
Mapping: Parses the text and matches it against official KTU codes and descriptions using semantic analysis.
Output: Returns a normalized, schema-ready object.
💎 Key Benefits
Compliance: Ensures every order matches official regulatory standards.
Data Integrity: Prevents duplicates and "dirty" entries in the procurement system.
Efficiency: Automates the tedious manual lookup of catalog codes.
